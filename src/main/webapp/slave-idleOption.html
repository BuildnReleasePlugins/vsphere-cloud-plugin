What should be done when the slave disconnects.
  • Shutdown: Shuts the virtual machine down, first attempting a Guest shutdown if VMware tools are installed, then a hard power off if VMware tools is not installed or the Guest shutdown fails.
  • Revert: Shuts down the virtual machine like Shutdown, then reverts. If a snapshot name is specified, the revert will be to that snapshot.
  • Suspend:: Suspends the virtual machine.
